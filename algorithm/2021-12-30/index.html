二叉树的最小深度

https://leetcode-cn.com/problems/minimum-depth-of-binary-tree

  • 如果 root 为空，直接返回 0
  • 如果 root 不空，没有左右子节点，返回 1
  • 如果 root 不空，且左右子树都有，返回 左右最小值 + 1
    • 否则返回单侧的最小值 + 1

这个题要和二叉树的最大深度进行对比，注意思路的区别。
